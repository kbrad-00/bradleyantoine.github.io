My key professional skills and focus areas.

Power BI Dashboards & Data Visualization

  • DAX measures for KPIs and trend analysis
  • Power Query for data cleaning and automation
  • Row-Level Security (RLS) for data governance
  • Dynamic filters and bookmarks for interactivity
  • Scheduled data refresh via Power BI Service
  • Custom color themes and layout optimization for accessibility

Transforming raw housing and financial data into reliable, queryable insights.

  • Using aggregate functions (SUM, AVG, COUNT) for reporting metrics
  • Optimizing query speed with indexes and execution plans
  • Designing stored procedures to automate recurring data exports
  • Handling data normalization and relationships for analytics integrity
  • Creating backup scripts and views for clean visualization pipelines
